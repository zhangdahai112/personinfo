常见问题解答

太平金融提供的遗产查询服务完全合法合规。我们要求委托人提供合法的授权文件和相关证明材料，通过正规渠道进行查询。所有查询过程严格遵守《个人信息保护法》等相关法律法规。

一般情况下需要提供：

  • 委托人身份证明
  • 被查询人身份证明（如已故需提供死亡证明）
  • 亲属关系证明
  • 遗产继承相关法律文件（如遗嘱等）
  • 授权委托书

具体材料可能因个案不同有所调整，我们的顾问会为您详细说明。

查询时间因个案复杂程度不同而有所差异。一般情况下，基础金融资产查询约需7-15个工作日；涉及房产、股权等复杂资产的全面查询可能需要20-30个工作日。我们会在受理委托时为您预估具体时间。

我们采用透明合理的收费模式：

  • 基础咨询免费
  • 查询服务按实际工作量收费，前期会提供明确报价
  • 成功查询到资产后收取合理比例的服务费（具体比例根据资产类型和金额确定）
  • 不成功不收费（部分特殊查询除外）
